📦 VARIABLES - CONCEPTO

📦 Las variables son como cajas con etiquetas

Donde guardamos información para usar después

🏷️ Etiqueta

Nombre de la variable

nombre

📦 Contenido

Valor que guardamos

"Ana"

🧪 DEMO: CREANDO VARIABLES

// Como una caja con nombre
let nombre = "Ana";
console.log(nombre);

// Cambiar el contenido de la caja
nombre = "Carlos";
console.log(nombre);

// Una caja que nunca cambia
const edad = 25;
console.log(edad);

🎭 TIPOS DE DATOS

JavaScript maneja 3 tipos principales:

  • String: Textos "Hola mundo"
  • Number: Números 25, 3.14
  • Boolean: Verdadero/Falso true, false

Cada tipo se comporta diferente

🧪 DEMO: STRINGS (TEXTOS)

let saludo = "¡Hola!";
let nombre = "María";
let mensaje = saludo + " " + nombre;
console.log(mensaje); // "¡Hola! María"

// Template literals (más fácil)
let mensajeFacil = `¡Hola! ${nombre}`;
console.log(mensajeFacil);

🧪 DEMO: NUMBERS (NÚMEROS)

let precio = 1500;
let descuento = 0.1;
let precioFinal = precio * (1 - descuento);
console.log(precioFinal); // 1350

// Operaciones básicas
console.log(10 + 5); // 15
console.log(10 - 5); // 5
console.log(10 * 5); // 50
console.log(10 / 5); // 2

🧪 DEMO: BOOLEANS (VERDADERO/FALSO)

let esMayorDeEdad = true;
let tieneLicencia = false;

console.log(esMayorDeEdad); // true
console.log(tieneLicencia); // false

// Comparaciones
console.log(18 >= 18); // true
console.log(5 > 10); // false
console.log("Ana" === "Ana"); // true

📋 ARRAYS - LISTAS

📋 Los arrays son como listas numeradas

Donde guardamos varias cosas relacionadas

❌ Sin Arrays

✅ Con Arrays

let colores = ["azul", "verde", "rojo"];

🧪 DEMO: TRABAJANDO CON ARRAYS

// Crear array
let coloresFavoritos = ["azul", "verde", "rojo"];

// Acceder a elementos (empezamos en 0)

coloresFavoritos.push("amarillo");
console.log("Tengo " + coloresFavoritos.length + " colores");

⚙️ FUNCIONES - CONCEPTO

👩‍🍳 Las funciones son como recetas

Les das ingredientes y te devuelven un resultado

📥 Ingredientes (Parámetros)

Lo que le das a la función

📤 Resultado (Return)

Lo que la función te devuelve

🧪 DEMO: FUNCIONES BÁSICAS

// Función simple
function saludar() {
    console.log("¡Hola mundo!");
}

// Llamar (ejecutar) la función
saludar(); // ¡Hola mundo!

// Función con parámetros
function saludarPersona(nombre) {
    console.log("¡Hola " + nombre + "!");
}

saludarPersona("Ana"); // ¡Hola Ana!

🧪 DEMO: FUNCIONES QUE DEVUELVEN VALORES

// Función que devuelve un valor
function sumar(a, b) {
    return a + b;
}

let resultado = sumar(5, 3);
console.log(resultado); // 8

// Usar directo
console.log("La suma es:", sumar(10, 20));

// Ejemplo práctico
function crearSaludo(nombre, edad) {
    return `Hola, soy ${nombre} y tengo ${edad} años`;
}

🤔 CONDICIONALES - DECISIONES

🛣️ Los condicionales son como decisiones

SI esto es verdad, HAZ esto
SI NO, haz esto otro

Permiten que el código tome decisiones automáticamente

🧪 DEMO: IF/ELSE

let edad = 20;

if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}

// Múltiples condiciones
let nota = 8;
if (nota >= 9) {
    console.log("Excelente");
} else if (nota >= 7) {
    console.log("Muy bueno");
} else {
    console.log("Necesitas estudiar más");
}

🔄 LOOPS - REPETICIÓN

🔁 Los loops son para hacer algo muchas veces

Sin escribir el mismo código una y otra vez

❌ Sin Loops

console.log("Hola 1");
console.log("Hola 2");
console.log("Hola 3");
console.log("Hola 4");
console.log("Hola 5");

✅ Con Loops

for (let i = 1; i <= 5; i++) {
  console.log("Hola " + i);
}

🧪 DEMO: FOR LOOP

// Contar del 1 al 5
for (let i = 1; i <= 5; i++) {
    console.log("Número:", i);
}

// Recorrer un array
let frutas = ["manzana", "banana", "naranja"];
for (let i = 0; i < frutas.length; i++) {
    console.log("Fruta:", frutas[i]);
}

// Sumar números
let suma = 0;
for (let i = 1; i <= 10; i++) {
    suma = suma + i;
}
console.log("Suma total:", suma);
