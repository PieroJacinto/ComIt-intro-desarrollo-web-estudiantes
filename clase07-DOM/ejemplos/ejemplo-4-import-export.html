📁 Separar Archivos JavaScript

Cómo poner tu JavaScript en archivos diferentes

🤔 ¿Por qué separar?

En lugar de tener todo en un archivo gigante, puedes separar tu código:

📂 ANTES (todo junto): mi-proyecto/ ├── index.html └── js/script.js ← TODO aquí (500 líneas!) 📂 DESPUÉS (separado): mi-proyecto/ ├── index.html ├── js/datos.js ← Solo datos (50 líneas) ├── js/funciones.js ← Solo funciones (100 líneas) └── js/main.js ← Solo eventos (50 líneas)

📄 Archivo 1: datos.js

// datos.js - Solo mis datos let frutas = ["manzana", "banana", "naranja"]; let miNombre = "Ana";

📄 Archivo 2: main.js

// main.js - Uso los datos del otro archivo function mostrarFrutas() { let lista = document.querySelector('#lista'); lista.innerHTML = ''; for (let i = 0; i < frutas.length; i++) { lista.innerHTML += `<li>${frutas[i]}</li>`; } } document.querySelector('#btn-mostrar').addEventListener('click', mostrarFrutas);

📄 HTML - Cargar ambos archivos

<!DOCTYPE html> <html> <body> <button id="btn-mostrar">Mostrar Frutas</button> <ul id="lista"></ul> <!-- Cargar archivos EN ORDEN --> <script src="js/datos.js"></script> <script src="js/main.js"></script> </body> </html>

🚀 Demo funcionando:

    ⚠️ Importante:

    • 📋 Orden importa: datos.js ANTES de main.js
    • 🔗 Las variables se comparten entre archivos
    • 🧹 Mantén cada archivo con un propósito
