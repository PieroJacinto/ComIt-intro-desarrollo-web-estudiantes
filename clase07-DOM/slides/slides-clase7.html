🔍 SELECCIONAR ELEMENTOS

querySelector: tu nueva herramienta favorita

// Seleccionar por ID const boton = document.querySelector('#mi-boton'); // Seleccionar por clase const lista = document.querySelector('.mi-lista'); // Seleccionar por etiqueta const titulo = document.querySelector('h1');

💡 Como CSS: # para ID, . para clase

✏️ CAMBIAR CONTENIDO

innerHTML: el poder de cambiar HTML

// Cambiar HTML completo elemento.innerHTML = '

¡Nuevo contenido!

'; // Con template strings (¡como en Clase 6!) const nombre = "Ana"; elemento.innerHTML = `

Hola ${nombre}

`; // Obtener valor de input const texto = input.value;

🖱️ EVENTOS: REACCIONAR AL USUARIO

addEventListener: escuchar clicks, teclas, etc.

// Escuchar click en botón boton.addEventListener('click', function() { console.log('¡Botón clickeado!'); }); // Escuchar escritura en input input.addEventListener('keyup', function(event) { const texto = event.target.value; console.log('Escribiste:', texto); });

🎯 Tu página responde al usuario

🎭 TIPOS DE EVENTOS

🖱️ Mouse:

  • click - Click del usuario
  • mouseover - Mouse encima
  • mouseout - Mouse sale

⌨️ Teclado:

  • keyup - Al soltar tecla
  • submit - Enviar formulario

💡 Cada evento tiene su momento perfecto
