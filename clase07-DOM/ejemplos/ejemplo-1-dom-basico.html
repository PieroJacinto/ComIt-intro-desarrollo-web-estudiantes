🎯 DOM Básico - Seleccionar y Modificar

Aprende a conectar JavaScript con HTML usando querySelector, innerHTML y style

🌐 ¿Qué es el DOM?

El DOM (Document Object Model) es la representación de tu HTML que JavaScript puede leer y modificar.

💡 Piénsalo así:

  • HTML = La estructura de tu casa
  • CSS = La decoración y colores
  • JavaScript + DOM = Los interruptores que cambian las luces, abren puertas, etc.

¡Finalmente vamos a hacer que nuestras funciones cambien lo que se ve en pantalla!

// 🔍 SELECCIONAR elementos const elemento = document.querySelector('#mi-elemento'); const lista = document.querySelector('.mi-lista'); // ✏️ MODIFICAR contenido elemento.innerHTML = '¡Nuevo contenido!'; elemento.textContent = 'Solo texto sin HTML'; // 🎨 MODIFICAR estilos elemento.style.color = 'blue'; elemento.style.backgroundColor = 'yellow';

✏️ Demo 2: Modificar Contenido

Cambia el contenido de los elementos con innerHTML y textContent.

📝 Contenido original que vamos a cambiar

🎨 Demo 3: Modificar Estilos

Cambia la apariencia de los elementos modificando sus estilos CSS desde JavaScript.

🎨 ¡Cambia mis estilos dinámicamente!
// CAMBIAR ESTILOS INDIVIDUALES elemento.style.color = 'red'; elemento.style.backgroundColor = 'yellow'; elemento.style.fontSize = '24px'; elemento.style.border = '3px solid blue'; // NOTA: CSS usa kebab-case pero JavaScript usa camelCase // CSS: background-color → JS: backgroundColor // CSS: font-size → JS: fontSize

🔑 Conceptos Clave del DOM

  • document.querySelector() - Selecciona el primer elemento que coincida
  • innerHTML - Cambia el contenido HTML interno
  • textContent - Cambia solo el texto (más seguro)
  • style - Modifica estilos CSS individuales
  • CSS → JavaScript: background-color → backgroundColor
